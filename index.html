종합비타민 성분 비교 분석 리포트

PCA(주성분 분석) 기법을 활용한 시판 제품의 포지셔닝 맵 구축

1. 탐구 개요 (Introduction)

본 프로젝트는 시중에서 판매되는 10종의 종합비타민 제품의 영양 성분 데이터를 수집하고, 주성분 분석(PCA) 기법을 적용하여 제품 간의 상관관계와 성분 배합의 경향성을 수학적으로 규명한 연구입니다.

2. 분석 방법 (Methodology)

파이썬(Python)의 Scikit-learn 라이브러리를 사용하여 8차원의 영양소 데이터를 2차원으로 축소하였습니다. 사용된 주요 코드는 다음과 같습니다.

# PCA 분석 핵심 코드
from sklearn.decomposition import PCA
from sklearn.preprocessing import StandardScaler

x_scaled = StandardScaler().fit_transform(x)
pca = PCA(n_components=2)
principalComponents = pca.fit_transform(x_scaled)

전체 코드는 여기(analysis.py)에서 확인할 수 있습니다.

Figure 1. PCA Biplot Analysis

분석 결과, PC1(가로축)은 '비타민 B/C의 함량 강도'를, PC2(세로축)는 '비타민 A vs 아연의 배합 성향'을 나타내는 것으로 확인되었습니다.

4. 결론 (Conclusion)

본 연구를 통해 시판 비타민 제품은 무작위로 설계되는 것이 아니라, '고함량 활력 부스팅'과 '기초 영양 밸런스'라는 두 가지 뚜렷한 설계 철학으로 구분됨을 데이터로 입증하였습니다.
